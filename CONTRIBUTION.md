# Contribution Guidelines

- Remember to add tests file
- Remember to update the README file.
- When updating the README file, ensure to increase the total number of problems.
- When updating the README file, add the problem to last in the list, don't add it somewhere in the middle of the category.
- Remember to update the CMakeLists.txt file if create new folder.
- Please check all existing files for patterns to follow for both solutions and tests.
- Time and space complexity always have to be this format: Time: O(n) | Space: O(1), not in two separate lines.
